# WordWolves2021
2021年度関東信越地区 冬季交流会アイスブレイクのために開発したワードウルフ運営用のDiscord Bot

# Build & Use

## Requirements
- Java 17

## Build
```bash
$ git clone git@github.com:5hrz/WordWolves2021.git
$ ./gradlew shadowJar
```

## Run
```bash
$ java -jar <Path to Jar File>.jar
```
